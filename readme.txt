WP HRM LITE is a powerful WordPress plugin to manage Human resource management needs of small and medium sized companies.

== Description ==

Using WP HRM LITE, companies can easily manage multiple Departments, Employees, Attendance, Notices, Holidays, Leave, Finance etc.

Step by step documentation is here : https://indigothemes.com/documentation/wphrm/

WP HRM Pro Features can be seen here : https://goo.gl/v17Ywu

== Installation ==

This section describes how to install the WP HRM LITE plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/wp-hrm-lite-human-resource-management-system` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP HRM LITE -> Settings screen to set your company settings and other settings.


== Frequently Asked Questions ==

= What is the documentation URL ? =

Documentation URL is https://indigothemes.com/documentation/wphrm/

= What is the main difference between WP HRM LITE and WP HRM PRO plugin ? =

WP HRM PRO is the advanced version of WP HRM LITE. WP HRM LITE has limitations that you can add only 5 employees but in WP HRM PRO, you can add unlimted employees. Other features like Salary Management, excel and pdf file download and Notifications Management will be available in WP HRM PRO version only.

= Why should I try WP HRM LITE ? =

If you're a small or medium sized company then you definitely want a management tool which can easily manage your admin and user side HRM needs. Using WP HRM LITE, you can add employees, departments, designations and mark the attendance for the same. You can also add various leave types and publish notices to employees.
